# Конструктор презентаций

Простой веб-конструктор презентаций на React + Vite + PptxGenJS.

## Что умеет

- создавать слайды;
- редактировать заголовок, подзаголовок и текст;
- выбирать тип слайда;
- выбирать тему оформления;
- сохранять проект в браузере;
- экспортировать проект в JSON;
- импортировать проект из JSON;
- скачивать готовую презентацию в `.pptx`.

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация на GitHub Pages

1. Создайте репозиторий `present`.
2. GitHub username уже указан: `svetlana18011991`.
3. Название репозитория уже указано в `vite.config.js`: `present`.
4. Выполните:

```bash
npm run build
npm run deploy
```

## Важно

Если репозиторий называется не `present`, измените:

```js
base: "/present/"
```

например:

```js
base: "/my-slides-app/"
```
