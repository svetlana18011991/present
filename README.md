# Конструктор презентаций

Веб-конструктор презентаций на React + Vite + PptxGenJS.

## Возможности

- создание и редактирование слайдов;
- выбор темы;
- общий фон презентации из своей картинки;
- отдельный фон для каждого слайда;
- картинка на каждом слайде;
- фоновая музыка для HTML/Genially;
- экспорт в PPTX;
- копирование HTML-кода для вставки в Genially;
- скачивание HTML-файла для вставки/публикации.

## Запуск локально

```bash
npm install
npm run dev
```

## Важно

Музыка добавляется в HTML-код для Genially. В PPTX музыка не встраивается этим конструктором.
Для стабильной работы используйте картинки до 5 МБ и музыку до 12 МБ.
